# Толдот р. Саадьи — תולדות ר׳ סעדיה

Статический сайт на GitHub Pages для книги **«Толдот р. Саадьи»** —
жизнеописания рабби Саадьи Либерова (авт. Мордехай-Цви Либеров, 5768/2008).

Каждая из 117 страниц показывает рядом: **изображение оригинального иврита** (справа) и **перевод на русский** (слева).

---

## Структура файлов

```
index.html          — Главная страница (оглавление + сетка всех страниц)
page.html           — Просмотр одной страницы (?p=N)
style.css           — Все стили
toc.json            — Оглавление с названиями глав
translations/
  ru/
    1.html          — Русский перевод страницы 1
    2.html
    …
    117.html
img/
  page-001.jpg      — Изображения страниц (150 dpi, JPEG)
  …
  page-117.jpg
original/
  Rabbi Saadya Liberow - 4 Adar I, 5768.pdf   — Исходный PDF
```

---

## Как добавлять / редактировать переводы

Откройте нужный файл в `translations/ru/`, например `translations/ru/5.html`.

Файл содержит HTML-разметку страницы. Если файл пустой — сайт покажет «Перевод для этой страницы ещё не добавлен».

После редактирования: `git commit + push` — сайт обновится автоматически.

---

## Как редактировать оглавление

Откройте `toc.json`. Добавьте запись в массив `chapters`:

```json
{
  "page": 15,
  "title_ru": "Глава 1. Детство в Рамене",
  "title_he": "ילדות ברומן"
}
```

Поле `title_he` необязательно. Главы отображаются в том порядке, в котором перечислены.

---

## Навигация по сайту

- **Стрелки влево/вправо** (клавиатура) — перелистывание страниц.
- **Поле с номером страницы** в шапке — прямой переход.
- **Адрес** вида `page.html?p=42` — прямая ссылка на страницу.

---

## Технический стек

- Чистый HTML + CSS + JS — никаких фреймворков, никакого npm.
- Работает полностью офлайн (открыть `index.html` напрямую).
- Lazy-loading изображений на главной; изображение текущей страницы загружается сразу.
